# くぼちゃんパン管理システム Ver.4.0.4

## Ver.4.0.4（今回の修正）
- スタッフ権限でも予約の削除ができるように変更（今までは管理者のみ）
- ホーム画面の更新が反映されない場合があったため、キャッシュバージョンを再度更新


## Ver.4.0.3（今回の追加機能）
- ホーム画面に「月・水・金の予約（直近）」カードを追加。予約受付日である直近の月・水・金曜日3日分の予約を、日付ごとに開閉できる一覧でまとめて確認できるようになりました。


## Ver.4.0.2（今回の修正）
- 権限パスワードがSupabaseと同期する前にログイン画面が出てしまい、端末ごとに古いパスワードが必要になる不具合を修正（起動時にSupabaseから先に読み込んでからログイン判定するように変更）
- 予約一覧・店頭販売売上の日付欄が日付をまたいでも自動で更新されない不具合を修正（日付が変わったら自動で今日の日付に更新）


Supabase共有に、簡易権限管理を追加した版です。

## 権限
- 管理者：すべて操作可能
- スタッフ：予約追加・編集、状況変更、売上入力
- 閲覧のみ：閲覧専用

## 初期パスワード
- 管理者：7870
- スタッフ：1111
- 閲覧のみ：0000

管理者でログイン後、設定画面の「権限管理」から変更してください。

## 注意
これは店舗内共有向けの簡易権限です。GitHub PagesのURL、Supabase URL、anon keyを外部に公開しない運用で使ってください。
より強く守る場合は、Supabase Auth + RLSでメールログイン方式にするのがおすすめです。
