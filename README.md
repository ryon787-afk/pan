# くぼちゃんパン管理システム Ver.4.0.1

Supabase共有に、簡易権限管理を追加した版です。

## 権限
- 管理者：すべて操作可能
- スタッフ：予約追加・編集、状況変更、売上入力
- 閲覧のみ：閲覧専用

## 初期パスワード
- 管理者：7870
- スタッフ：1111
- 閲覧のみ：0000

管理者でログイン後、設定画面の「権限管理」から変更してください。

## 注意
これは店舗内共有向けの簡易権限です。GitHub PagesのURL、Supabase URL、anon keyを外部に公開しない運用で使ってください。
より強く守る場合は、Supabase Auth + RLSでメールログイン方式にするのがおすすめです。
