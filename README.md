# パン屋予約・売上管理アプリ

スマホのブラウザで使える、パン屋向けの予約・売上管理PWAです。

## ファイル構成

- `index.html` 画面本体
- `style.css` デザイン
- `app.js` アプリの処理
- `manifest.json` ホーム画面追加用の設定
- `service-worker.js` オフライン対応用
- `icons/` アプリアイコン

## 使い方

1. GitHubにこのフォルダの中身をアップロードします。
2. GitHub Pagesを有効にします。
3. 発行されたURLをスマホで開きます。
4. ブラウザの共有メニューから「ホーム画面に追加」を選びます。

## 主な機能

- 商品登録
- 商品価格登録
- 予約登録
- 予約金額の自動計算
- 日別売上保存
- 天気入力
- 備考欄
- 月売上・年売上・月平均売上
- 商品別予約実績
- 天気別売上
- バックアップ書き出し・復元

## データ保存

データは端末のブラウザ内、LocalStorageに保存されます。
スマホ変更やブラウザ削除に備えて、定期的にバックアップ書き出しをしてください。
